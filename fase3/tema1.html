Elementos semanticos en HTML


Una de las principales ventajas de HTML5 es la inclusión de elementos semánticos, o marcados semánticos, que nos ayudan a definir las distintas divisiones de una página web. En las versiones anteriores a HTML5 se solía utilizar la etiqueta "div" para realizar las divisiones de una página web, pero actualmente es recomendado utilizar los elementos semánticos ya que describen claramente su propósito. Como se puede observar en el siguiente esquema, mediante marcadores semánticos se puede diferenciar claramente el contenido que referencia cada etiqueta.
